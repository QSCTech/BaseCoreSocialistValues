## CLI

We provide a CLI program named `bcsv` to encode and decode conveniently.

### Install

With Rust's package manager `cargo`, you can install `bcsv` via:

```
cargo install bcsv
```

### Usage

To encode a file:

```
bcsv [filename]
```

To decode a file:

```
bcsv -d [filename]
```

With no filename or filename is `-`, it will read from standard input.
